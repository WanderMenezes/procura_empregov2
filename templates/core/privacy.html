Política de Privacidade

Última atualização: 12 de Fevereiro de 2026

1. Introdução

A Base Nacional de Jovens está comprometida em proteger a privacidade dos utilizadores da nossa plataforma. Esta política descreve como recolhemos, usamos e protegemos os teus dados pessoais.

2. Dados que Recolhemos

Podemos recolher os seguintes tipos de informação:

  • Informações pessoais: Nome, data de nascimento, sexo, contactos
  • Informações educacionais: Nível de educação, área de formação, instituição
  • Informações profissionais: Experiência, skills, interesses
  • Documentos: CV, certificados (opcional)

3. Como Usamos os Teus Dados

Os teus dados são usados para:

  • Criar e gerir o teu perfil na plataforma
  • Conectar-te com empresas e oportunidades
  • Enviar notificações sobre vagas e candidaturas
  • Gerar relatórios estatísticos (dados anonimizados)

4. Partilha de Dados

Os teus dados pessoais só são partilhados com empresas quando:

  • Autorizas explicitamente no teu perfil
  • Uma empresa solicita contacto e o administrador aprova
  • É exigido por lei ou autoridade competente

5. Os Teus Direitos

Tens o direito de:

  • Aceder aos teus dados pessoais
  • Corrigir informações incorretas
  • Eliminar a tua conta e dados
  • Tornar o teu perfil invisível
  • Retirar o consentimento de uso de dados

6. Segurança

Implementamos medidas de segurança técnicas e organizacionais para proteger os teus dados contra acesso não autorizado, alteração, divulgação ou destruição.

7. Retenção de Dados

Os teus dados são mantidos enquanto a tua conta estiver ativa. Podes solicitar a eliminação da tua conta a qualquer momento.

8. Contacto

Para questões sobre privacidade, contacta-nos através de info@basenacionaljovens.st.
